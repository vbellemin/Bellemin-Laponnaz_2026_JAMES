# Bellemin-Laponnaz_2026_JAMES
Sources for the paper Bellemin-Laponnaz et al. A variational method for reconstructing and separating balanced motions and internal tides from wide-swath altimetric sea surface height observations. *ESS Open Archive*, 2025. This repository, along with the SEANOE datasets can be used to reproduce the experiments and the analyzes of the manuscript.
